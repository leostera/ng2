a minimalistic modular angularjs app generator

motivation none of the tools out there help you build big sized angularjs apps with a proper modular approach

installation ng2 can be installed with npm doing

> npm -g i ng2

start normally you'd work with the generators, but when starting this is what you'd do

> ng2 start myProject
> cd myProject
> component install
# we'll talk about component in a minute
> ng2 server
# check your running sample at localhost:9000

usage the generators allow you to scaffold angular resources quickly and setup the surroundings automatically for you

module <username/name>    scaffold a module
controller <name>         scaffold a controller
directive <name>          scaffold a directive
filter <name>             scaffold a filter
provider <name>           scaffold a provider
service <name>            scaffold a service
view <name>               scaffold a view
style <name>              scaffold a stylesheet

then there's some helper methods 

build                     build the project
watch                     continuously watch and build
server                    watch and run a webserver

finally all your modules are component.io-compatible commonjs modules, put them up at github and people will be able to install easily. like my port of angularjs

> component install leostera/angular.js
